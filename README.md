# Traffic Flow

Next.js (App Router) app with TypeScript and Tailwind CSS v4.

## Scripts

- `npm run dev` — development server  
- `npm run dev:turbo` — dev with Turbopack  
- `npm run build` / `npm run start` — production  
- `npm run lint` — ESLint  

See `PROJECT_STRUCTURE.md` for folder layout and conventions.
